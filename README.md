# MiniBankSystem

A simple console-based banking system in Java that allows multiple customers to manage their accounts. Users can check their balance, deposit money, withdraw money, and view their transaction history.

---

## Features

- Add multiple customers with unique account numbers.
- Check account balance.
- Deposit money into the account.
- Withdraw money from the account (with insufficient funds check).
- View transaction history for each customer.
- Menu-driven console interface.

---

## Technologies Used

- Java
- Arrays for storing customer data and transaction history
- Scanner for user input
- Switch-case and loops for menu handling

---

## How to Run

1. Clone or download the repository.
2. Open the project in your preferred IDE (Eclipse, IntelliJ, or VS Code with Java support).
3. Run the `MiniBank.java` file.
4. Follow the on-screen instructions to use the banking system.

## Project Level

This project is suitable for **beginner to early-intermediate Java learners**. It demonstrates:

- Array manipulation
- Loops and conditional statements
- Menu-driven applications
- Basic input validation and transaction history tracking
